##  Features

- Add new invoices  
- Edit existing invoices  
- Delete invoices  
- View all invoices in a clean table  
- Server-side validation  
- Authentication (Login & Register using Laravel UI)  
- Simple and clean Blade views  

---

##  Tech Stack

- **Laravel 9**  
- **Laravel UI (Bootstrap Auth)**  
- **MySQL**  
- **Blade Templates**  

---

##  Database Fields

Each invoice contains:

- `invoice_name`  
- `amount`  
- `due_date`
